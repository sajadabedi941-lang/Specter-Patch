Specter Compatible Patch - Phase 1
==================================

Purpose
-------
Create a clean overlay folder that adds new playable faction templates
without modifying original Specter BIG/ZIP files.

Factions (Iraq-copy method)
---------------------------
- Turkey
- Ukraine
- Pakistan
- Saudi Arabia (Side=SaudiArabia)
- UAE
- India
- Japan

Layout
------
patch/Data/INI/... is intended to be packed as a GenLauncher addon/patch
or loaded as loose Data override on top of Specter.

Phase 1 includes
----------------
- PlayerTemplate stubs for each new Side
- ControlBarScheme stubs (Iraq chrome, new Side)
- Science root stubs (reuse SCIENCE_Iraq for now, like NorthKorea)
- CommandSet stubs for each Command Center
- Object/Specter/<Country>/ directory tree mirroring Iraq Army
- Placeholder CommandCenter.ini per country (renamed Iraq object)

Not included yet
----------------
- Full building/unit/aircraft Object copies
- CommandButton definitions
- CSF string table entries
- Unique art/audio

Dynamic Pricing System (global)
-------------------------------
Reusable INI framework under Data/INI/Economy/:
  FactionEconomy, EquipmentOrigin, TechnologyClass, AsymmetricBalance,
  DomesticProduction, ResourceIncome, PricingDefaults, UnitPricingRegistry
Formula: BaseCost × Economy × Origin × Tech × Asymmetric × Domestic
Generator: tools/economy/apply_dynamic_pricing.py (idempotent via PatchBaseCost)
Applied to Turkey tree + Advanced Air Base shells; future factions auto-use tables.
Resource income: SupplyCenter AutoDeposit scaled by EconomyRating.
Domestic MIC: Turkey_MIC grants Upgrade_Patch_DomesticIndustry (−5% VEHICLE/INFANTRY).

Country Balance System (central INI)
------------------------------------
Single source of truth: Data/INI/CountryBalance.ini
  - Per-country Unit/Aircraft/Weapon/Upgrade cost multipliers
  - Port/oil/trade/supply income multipliers
  - Global Origin/Tech/Category/Asymmetric/Domestic rules (no per-unit hardcoding)
  - Applicator: tools/economy/apply_country_balance.py (deterministic bake for MP)
  - Legacy Economy/*.ini tables superseded; do not edit for live balance

Multiplayer desync prevention
-----------------------------
Gate document: patch/SYNC_CHECKLIST.md
Audit tool: tools/economy/sync_audit.py (must PASS before new units)
Rules: deterministic bake, no archive edits, unique IDs, no player→*_AI production,
identical patch package on all clients.
