SPECTER FINAL BOOT-READY PATCH
==============================

Ready-to-use Patch overlay for Specter Ultimate Warfare Expansion.
No installer scripts. Copy files into the game and play.

INSTALL
-------
1. Close Generals Zero Hour / Specter.
2. Copy/merge:
     Patch\Data\   into   <GameRoot>\Data\
     Patch\Art\    into   <GameRoot>\Art\
3. Launch Specter.

This overlay includes repaired Object\Specter INIs and related Data\INI
patch files (weapons, command sets, buttons, faction objects, etc.).

Do not extract New folder.zip flat into Data\INI\.
Those objects belong under Data\INI\Object\Specter\... (already correct here).

See VALIDATION_REPORT.txt for scan counts.
